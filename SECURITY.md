# Security Policy

Dieses Projekt ist eine reine Browser-App ohne Backend.

## Unterstützte Versionen

Aktuell wird die neueste Version auf `main` gepflegt.

## Sicherheitsproblem melden

Bitte keine echten API-Keys, Kundendaten oder vertraulichen Druckdateien in öffentlichen Issues posten.

Wenn du ein Sicherheitsproblem findest, erstelle ein Issue mit einer allgemeinen Beschreibung oder kontaktiere den Repository-Besitzer privat über GitHub.

## Datenschutz-Hinweis

Die App verarbeitet Bilder lokal im Browser. Es findet kein Upload auf einen Projektserver statt.
